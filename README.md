# Курсовая работа: Клиент к веб-службе ЦБ РФ

## Описание
Проект реализует прокси-сервер, который получает данные о курсах валют с официальной SOAP-службы ЦБ РФ и предоставляет их пользователю в формате JSON.

## Структура
- `proxy-server.js` — SOAP-прокси (слушает на порту 3001)
- `user-client.js` — клиент для тестирования
